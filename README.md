# 📰 Google News 進階檢索與 Excel 匯出系統

這是一個基於 **Python Flask** 開發的輕量化新聞輿情檢索工具。透過 Google News RSS 介面，本系統支援強大的原生搜尋指令與日期篩選功能，並能將查詢結果自動整理、排序，最終一鍵匯出為結構化的 Excel 報表。

---

## ✨ 核心功能

* **進階語法支援**：直接在關鍵字欄位輸入 Google 搜尋運算子（如 `intitle:`、`site:`、`-`、`OR` 等）。
* **精準時間過濾**：提供開始與結束日期選擇，自動對接 Google `after:` 與 `before:` 參數。
* **新舊自動排序**：搜尋結果預設按「發佈時間」**從新到舊**排列，方便掌握最新動態。
* **格式化時間輸出**：將原始 RSS 時間格式統一轉化為 `YYYY-MM-DD HH:MM:SS`（年-月-日 時:分:秒）。
* **智慧 Excel 命名**：匯出的檔案名稱會自動組合「關鍵字」與「日期範圍」，格式為：`新聞匯出_關鍵字_2025-01-01_至_2026-01-09.xlsx`。
* **資料清洗**：自動過濾 HTML 標籤，確保匯出的摘要內容純淨無雜質。
* **內網相容性**：內建 SSL 驗證豁免機制，確保在嚴格的辦公室防火牆環境下也能正常抓取。

## 💡 Google 搜尋指令說明 (直接輸入搜尋框)

本系統支援所有 Google 原生搜尋指令，您可以靈活組合各種條件：

| 指令名稱 | 範例 | 效果說明 |
| :--- | :--- | :--- |
| **" "** (雙引號) | `"沙崙農場"` | **精準搜尋：** 必須完全符合括號內的字串與順序。 |
| **-** (減號) | `沙崙農場 -光電` | **排除關鍵字：** 搜尋結果中不得出現減號後的詞彙。 |
| **OR** (或) | `台南 OR 高雄` | **聯集搜尋：** 搜尋包含「台南」或「高雄」其中之一的新聞。 |
| **intitle:** | `intitle:沙崙農場` | **限定標題：** 只在新聞標題中尋找該關鍵字。 |
| **site:** | `沙崙 site:ltn.com.tw` | **指定網站：** 只搜尋特定媒體（如自由時報）。 |
| **( )** (小括號) | `(台南 OR 高雄) 沙崙` | **邏輯分組：** 搜尋包含「沙崙」且包含「台南」或「高雄」的新聞。 |
| **AROUND(n)** | `沙崙 AROUND(5) 抗議` | **距離搜尋：** 兩關鍵字之間隔不得超過 n 個字。 |

## 🚀 快速安裝與開發

### 1. 本地環境建置
請確認您的電腦已安裝 Python 3.8+。

```bash

python -m venv .venv

# Windows 啟動:
.\.venv\Scripts\activate

source .venv/bin/activate
